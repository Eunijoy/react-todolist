# React TypeScript Todo List

A clean and beginner-friendly **Todo List application built with React and TypeScript**, designed to focus on real fundamentals instead of over-engineering.  
This project lives inside a parent folder named **`react/`** and serves as a solid foundation for understanding state management, component structure, and basic UI/UX principles in React.

---

## Project Structure

react/
└── todo-list/
├── src/
│ ├── App.tsx
│ ├── TodoInput.tsx
│ ├── TodoItem.tsx
│ ├── App.css
│ └── main.tsx
└── README.md


---

## Features

- Add new tasks
- Mark tasks as completed using a checkbox
- Delete tasks
- Clean and modern UI
- Responsive layout
- Built with React + TypeScript
- Beginner-friendly and easy to extend

---

## Tech Stack

- React
- TypeScript
- CSS (no UI libraries)
- Vite or Create React App
